# Getting Started

## Installation

To get started with this project, follow these steps:

1. Clone the repository
2. Install the requirements:

```bash
pip install -r requirements.txt
```

3. Run MkDocs:

```bash
mkdocs serve
```

4. Open your browser and go to `http://localhost:8000`
